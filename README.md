# 🚀 Curso Impressionador - Hashtag Treinamentos

Este repositório foi criado para acompanhar minha evolução durante o **Curso Impressionador** da Hashtag Treinamentos.

O objetivo é centralizar os projetos, exercícios e estudos realizados ao longo do curso, explorando diferentes áreas da programação aplicada a dados e automação.

---

## 📚 Sobre o curso

O Curso Impressionador aborda, de forma prática, diversas áreas muito utilizadas no mercado de tecnologia e negócios, incluindo:

- ⚙️ Automação de tarefas
- 📊 Análise de dados
- 🌐 Desenvolvimento Web
- 🤖 Machine Learning e previsões
- 🧠 Aplicações práticas com Python

Durante o curso, cada módulo gera projetos e estudos que serão adicionados progressivamente neste repositório.

---

## 🎯 Objetivo do repositório

Este repositório **não tem como foco projetos finais complexos**, mas sim:

- Consolidar aprendizado na prática
- Registrar minha evolução técnica
- Servir como material de consulta futura
- Demonstrar consistência nos estudos
- Aplicar conceitos em projetos reais e incrementais

---

## 🛠️ Tecnologias e ferramentas utilizadas

Ao longo do curso, serão utilizadas tecnologias como:

- Python
- Pandas
- Numpy
- Matplotlib / Seaborn
- Automação com Python
- Web frameworks (quando aplicável)
- Machine Learning (Scikit-learn e similares)
- Jupyter Notebook
- Git & GitHub
